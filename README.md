API with Vanilla
===

> Example of a Node.js API without framework.


Status
---

> Runnable


Getting started
---

Install the dependencies
```
$ npm install
```

Run the tests
```
$ npm test
```

Start the project
```
$ npm start # default port is 3000
```


Features
---

Based on the [Katapi](https://github.com/octo-woapi/katapi) kata!
